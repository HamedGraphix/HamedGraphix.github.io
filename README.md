# Mohamed Hamed Portfolio

A professional portfolio website for Mohamed Hamed, a premium graphic designer specializing in high-end visual identities, 3D renders, and creative digital solutions.

## Features

- **Dynamic Portfolio Showcase**: Categorized sections for Branding, Photo Manipulation, Posters, and 3D Renders.
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Modern UI/UX**: Features a sleek dark mode aesthetic with custom cursor effects and smooth transitions.
- **Interactive Animations**: Powered by AOS (Animate On Scroll) for engaging visual effects.
- **Direct Contact Integration**: Functional contact form with mailto integration and direct social media links (WhatsApp, Instagram, TikTok).

## Technologies Used

- **Core**: HTML5, CSS3, JavaScript (Vanilla)
- **Libraries**: 
  - [AOS](https://michalsnik.github.io/aos/) (Animate On Scroll)
  - [SwiperJS](https://swiperjs.com/) (Touch Slider)
- **Icons**: [Font Awesome](https://fontawesome.com/)
- **Typography**: [Google Fonts](https://fonts.google.com/) (Inter, Syne)
